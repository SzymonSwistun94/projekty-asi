### Szymon Świstun

- [link do wdrożonej na zewnętrznym serwerze aplikacji](https://proj1-ror.herokuapp.com/)

* ruby 2.3.3
* rails 5.0.2
* Postgresql

Strona służy do przechowywania i udostępniania notatek.

* materialize-sass - wygląd,
* devise - logowanie,
* redcarpet - parser Markdown.
